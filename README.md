# lab02-mandelbrot-team-emily-mathias
lab02-mandelbrot-team-emily-mathias created by GitHub Classroom
  
This project was created as part of a school project to learn C++ using the Mandelbrot set.

In this program, coordinates are chosen in a predefined scale and are used to color a square of a certain color by computing their iteration in the Mandelbrot set.

The user also has the possibility to navigate through the Mandelbrot set with the keyboard
